Lesson 4: Working with Lists

Lists are a type of data structure

They are self explanatory, they store a list of items

Lists are specified with square brackets

Ex. 

  mylist = [1, 2, 3, 4, 5]


Lists can store: Strings, other lists, dictionaries, integers, variables

Lists with lists inside them are called: Nested Lists

Basic list methods 1. Adding an item to a list:

mylist = []

mylist.append('hello')

print(mylist)

>>> ['hello'] 2. Deleting items from a list 

  mylist = ['hello']
  


  mylist.pop(0)
  



  print(mylist)
  >>> []

You may be thinking what the 0 in the pop function is.



0 is the index of the item that you want to delete from the list.



List indecies start with 0. So the first item will have an index of 0, the second will have 1, and so on.

There is another method to delete items from a list.

This is using the del keyword



  mylist = ['hello']
  


  del mylist['hello']
  


  print(mylist)
  


  >>>[]
